**GET** `/chats/bookmarks/`

This endpoint fetches **all bookmarked messages** across all chat sessions for the authenticated user.  
It returns an array of message objects, each representing a bookmarked chat message.

A valid authentication token is required.

# ğŸ§© Bookmarked Message Object Structure

Each element in the returned array has the following fields:

| Field           | Datatype     | Required | Description |
|------------------|--------------|----------|-------------|
| message_uuid     | string       | Yes      | Unique message identifier |
| session_uuid     | string       | Yes      | ID of the chat session the message belongs to |
| role             | string       | Yes      | `"user"` or `"assistant"` |
| content          | string       | Yes      | Text content of the message |
| structured_data  | string/null  | No       | Additional metadata (optional) |
| timestamp        | string       | Yes      | Timestamp of message creation |
| is_bookmarked    | boolean      | Yes      | Always `true` for this endpoint |
| attachment       | array        | Yes      | List of attachments (empty if none) |
