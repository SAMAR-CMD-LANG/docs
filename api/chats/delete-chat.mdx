---

**DELETE** `/chats/sessions/{session_uuid}/`

This endpoint permanently deletes a **chat session** belonging to the authenticated user.  
When a session is deleted:

- The chat session itself is removed  
- All messages inside that session are also deleted  

This action cannot be undone.

The endpoint requires only the **Authorization token**.  
There is **no request body**.
