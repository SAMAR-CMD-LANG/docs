# GET `/analytics`

Fetches analytics for the authenticated user, including journals, notes, and tasks.  
Token authentication required.

---

## üîê Required Headers

| Key | Value Format | Required | Description |
|------|-------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the user |

---

## üì• Response ‚Äî 200 OK

### Top-Level Fields

| Field | Type | Description |
|--------|--------|-------------|
| activities | Object | Contains all analytics sections |

---

## üìä Journal Analytics

| Field | Type | Description |
|--------|--------|-------------|
| total_entries | Number | Total journal entries |
| time_breakdown | Object | Count of entries per time period |

### time_breakdown Fields

| Field | Type | Description |
|--------|--------|-------------|
| last_month | Number | Entries created last month |
| last_week | Number | Entries created last week |
| this_week | Number | Entries created this week |

---

## üìù Notes Analytics

| Field | Type | Description |
|--------|--------|-------------|
| total_pages | Number | Total number of note pages |
| category_breakdown | Object | Entries per category |

### category_breakdown Example

> Values vary per user

- Work  
- Personal  
- Academics  
- Health & Fitness  
- Finance  
- Social & Events  
- Ideas & Creativity  
- Home & Lifestyle  
- Others  

---

## ‚úÖ Tasks Analytics

| Field | Type | Description |
|--------|--------|-------------|
| total_tasks | Number | Total number of tasks |
| status_breakdown | Object | Count per task status |
| priority_breakdown | Object | Count per priority level |
