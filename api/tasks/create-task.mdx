### **POST /tasks/tasks/**

Creates a new task inside the specified list. Supports notes, due dates, start times, and reminders.

---

## **Headers**

| Key | Value | Required | Description |
|------|--------|----------|-------------|
| Authorization | Token `<token>` | Yes | User authentication token |
| Content-Type | application/json | Yes | JSON request body |

---

## **Request Body Fields**

| Field | Type | Required | Description |
|-------|--------|----------|-------------|
| title | string | Yes | Title of the task |
| description | string | No | Optional description of the task |
| target_list_uuid | string | Yes | UUID of the list where the task will be created |
| start_date | string | No | Task start date (YYYY-MM-DD) |
| start_time | string | No | Task start time (HH:MM:SS) |
| due_date | string | No | Task due date (YYYY-MM-DD) |
| due_time | string | No | Task due time (HH:MM:SS) |
| reminders | array | No | List of reminder objects |
| reminders[].actual_trigger_time | string | No | Exact time to trigger the reminder |
| reminders[].offset_preset | string | No | Preset such as `"at_due_time"` |

---

## **Request Example**

```json
{
  "title": "work",
  "description": "",
  "target_list_uuid": "02696658-b3ef-4175-8619-321ad773e011",
  "start_date": "2025-08-25",
  "start_time": "07:00:00",
  "due_date": "2025-08-25",
  "due_time": "12:30:00",
  "reminders": [
    { "actual_trigger_time": "2024-12-05T13:30:00Z" },
    { "offset_preset": "at_due_time" }
  ]
}
```

## **Response Fields**

| Field        | Type        | Description                    |
|--------------|-------------|--------------------------------|
| uuid         | string      | Unique identifier for the task |
| title        | string      | Title of the task              |
| description  | string      | Description text               |
| list_uuid    | string      | Parent list UUID               |
| section_uuid | string/null | Section UUID if assigned       |
| status       | string      | Task status                    |
| start_date   | string      | Task start date                |
| start_time   | string      | Task start time                |
| due_date     | string      | Due date                       |
| due_time     | string      | Due time                       |
| order        | number      | Ordering index                 |
| priority     | string/null | Priority level                 |
| is_archived  | boolean     | Whether archived               |
| reminders    | array       | List of attached reminders     |
| created_at   | string      | Creation timestamp             |
| updated_at   | string      | Update timestamp               |
