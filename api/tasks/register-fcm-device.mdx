# POST `/fcm/register/`

Register a device for Firebase Cloud Messaging notifications.  
Requires an authentication token.

---

| Key | Value Format | Required | Description |
|------|-------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the user |

---

## üì§ Request Body

| Field | Type | Required | Description |
|--------|--------|----------|-------------|
| registration_token | String | Yes | Firebase registration token for the device |
| device_type | String | Yes | Device type (`WEB`, `ANDROID`, `IOS`) |
