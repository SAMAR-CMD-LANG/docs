# POST `/notes/`

Create a new note.  
Supports text notes, file attachments (images, documents, audio), and optional audio transcription.

---

## üîê Required Headers

| Key | Value Format | Required | Description |
|------|--------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the user |
| Content-Type | multipart/form-data | Yes | Required for uploading files |

---

## üì§ Form Data Fields

| Field      | Type     | Required | Description |
|------------|----------|----------|-------------|
| title      | String   | No       | Title of the note (default: "Untitled Note") |
| category   | Integer  | No       | Category ID |
| transcribe | Boolean (`true/false`) | No | If uploading audio, `transcribe=true` enables transcription |
| file       | File     | No       | Optional attachment (image, document, or audio file) |

---

## üì• Response ‚Äî 201 Created

### Response Fields

| Field            | Type         | Description |
|------------------|--------------|-------------|
| note_uuid        | UUID         | Unique identifier of the created note |
| title            | String       | Title of the note |
| category         | Integer      | Category assigned to the note |
| labels           | Array        | Labels applied to the note |
| is_pinned        | Boolean      | Whether the note is pinned |
| is_public        | Boolean      | Whether the note is public |
| content          | String       | Text content of the note |
| attachments      | Array        | Uploaded files metadata |
| ai_content_items | Array        | AI-generated content items |
| created_at       | ISO datetime | Timestamp of creation |
| updated_at       | ISO datetime | Timestamp of last update |
