**GET** `/notes/{note_uuid}/`

This endpoint retrieves the **full details of a specific note** identified by its `note_uuid`.  
The response includes:

- Full HTML content  
- Labels  
- Pin status  
- Attachments (parsed from the HTML)  
- AI-generated content (if any)  
- All metadata (timestamps, visibility flags)

Only the authenticated user can view their own notes.

# üß© Response Body Fields

| Field            | Datatype      | Required | Description |
|------------------|---------------|----------|-------------|
| note_uuid        | string        | Yes      | Unique identifier of the note |
| title            | string        | Yes      | Title of the note |
| category         | number        | Yes      | Category ID |
| labels           | array         | Yes      | Array of label IDs |
| is_pinned        | boolean       | Yes      | Whether the note is pinned |
| is_public        | boolean       | Yes      | Whether the note is public |
| content          | string (HTML) | Yes      | Full HTML note content |
| attachments      | array         | Yes      | Attachments parsed from HTML |
| ai_content_items | array         | Yes      | AI-generated items (if any) |
| created_at       | string        | Yes      | Timestamp of creation |
| updated_at       | string        | Yes      | Timestamp of last update |
