title: "Create Note (Audio Attachment)"

---

**POST** `/notes/`

Creates a new note with an audio attachment.  
Supports optional transcription if `transcription=true`.  
Returns full note metadata including embedded audio HTML.

---

| Key           | Value Format         | Required | Description |
|---------------|-----------------------|----------|-------------|
| Authorization | Token `<token>`         | Yes      | Authentication token |
| Content-Type  | multipart/form-data   | Yes      | Required for audio uploads |

| note_uuid    | string  | Yes | Client-generated UUID for the note |
| title        | string  | Yes | Title for the note |
| category     | number  | Yes | Category ID |

## üì¶ Response Fields

| Field | Type | Description |
|--------|------|-------------|
| note_uuid | string | Unique ID for the note |

| category | number | Category ID |
| labels | array | List of labels |
| is_pinned | boolean | Pin status |
| is_public | boolean | Public accessibility |

| updated_at | string | Last updated timestamp |
