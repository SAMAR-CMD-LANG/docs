**POST** `/notes/`

Creates a new note containing **text/HTML content** plus **image attachments**.  
The request must be sent as **multipart/form-data**.  
Images will be uploaded and included inside the note‚Äôs item as HTML `<attachment-embed>` blocks.

---

## üîê Required Headers

| Key           | Value Format           | Required | Description |
|--------------|-------------------------|----------|-------------|
| Authorization | `Token <token>`        | Yes      | User authentication token |
| Content-Type  | `multipart/form-data`  | Yes      | Required for file uploads |

---

## üß© Form-Data Fields

| Field       | Datatype | Required | Description |
|-------------|----------|----------|-------------|
| note_uuid   | string   | Yes      | UUID for the new note |
| item_uuid   | string   | Yes      | UUID for the note item being created |
| title       | string   | Yes      | Title of the note |
| category    | number   | Yes      | Category ID assigned to the note |
| content     | string   | Yes      | HTML/text content for the note item |
| file        | file     | Yes      | One or more image attachments (`png`, `jpg`, `jpeg` etc.) |

---

## üìÆ Example cURL

```bash
curl -X POST "https://your-api.com/notes/" \
  -H "Authorization: Token <token>" \
  -F "note_uuid=11ea8512-a72d-4868-9eff-ccefad1659b0" \
  -F "item_uuid=6ca9f5c3-cd88-490d-a7bd-444c171dbea9" \
  -F "title=My Note Title" \
  -F "category=2" \
  -F "content=This is my note content" \
  -F "file=@image1.jpg" \
  -F "file=@image2.png"
```

# üì¶ Response Body Fields

| Field            | Datatype | Required | Description |
|------------------|----------|----------|-------------|
| item_uuid        | string   | Yes      | Unique identifier of the created note item |
| content          | string   | Yes      | Text/HTML content including embedded image HTML |
| order            | number   | Yes      | Position of the item inside the note |
| attachment       | array    | Yes      | List of uploaded image metadata |
| ai_content_items | array    | Yes      | AI-generated enrichments (if any) |
| created_at       | string   | Yes      | Timestamp of creation |
| updated_at       | string   | Yes      | Timestamp of last update |

# üìé Attachment Object Structure

| Field       | Datatype | Required | Description |
|-------------|----------|----------|-------------|
| id          | number   | Yes      | Attachment ID |

| size        | number   | Yes      | File size in bytes |
| name        | string   | Yes      | Original file name |
| created_at  | string   | Yes      | Upload timestamp |
