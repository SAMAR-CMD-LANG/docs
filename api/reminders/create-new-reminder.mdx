# POST `/tasks/tasks/{task_uuid}/reminders/`

Create one or more reminders for a task.  
Each array item must include either an `offset_preset` or an `actual_trigger_time`.

---

## üîê Required Headers

| Key | Value Format | Required | Description |
|-----|--------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the user |

---

## üîó Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| task_uuid | UUID | Yes | ID of the task to attach reminders to |

---

## üì§ Request Body

Array of reminder objects:

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| offset_preset | String | Optional | A preset offset such as `"10_minutes_before"` or `"1_hour_before"` |
| actual_trigger_time | ISO datetime | Optional | Exact trigger time in UTC |

### Example Request Body

```json
[
  {"offset_preset": "10_minutes_before"},
  {"offset_preset": "1_hour_before"},
  {"actual_trigger_time": "2024-01-15T10:00:00Z"}
]
```

## Response Fields
| Field   | Type   | Description                 |
|---------|--------|-----------------------------|
| detail  | String | Summary message             |
| created | Array  | List of created reminders   |

Below are the fields returned for each reminder object.

| Field               | Type         | Description                                        |
|---------------------|--------------|----------------------------------------------------|
| uuid                | UUID         | Unique identifier of the reminder                  |
| offset_preset       | String/null  | Offset preset or null if using `actual_trigger_time` |
| actual_trigger_time | ISO datetime | When the reminder will trigger                     |
| is_active           | Boolean      | Whether the reminder is active                     |
