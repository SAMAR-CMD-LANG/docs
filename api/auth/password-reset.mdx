title: "Request Password Reset"
description: "Send a password reset email to a registered user containing a reset token."
---



**POST** `/auth/password-reset/request/`

This endpoint sends a password reset email to the user.  
The email contains a **reset token** inside a secure link.  
The user must click that link to continue the password reset process and set a new password.

This endpoint always returns a generic success message for security purposes, even if the email does not exist in the system.
