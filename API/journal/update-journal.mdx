title: "Update Journal Content"
description: "Updates only the text content of an existing journal entry while keeping attachments unchanged."
---

**PATCH** `/journals/{journal_uuid}/`

This endpoint updates **only the journal content** of an existing journal entry.  
Attachments such as images or audio remain unchanged.

Only the authenticated user can update their own journals.
