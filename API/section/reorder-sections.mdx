---
title: Reorder Sections
---

### **PATCH /tasks/lists/{list_uuid}/sections/reorder/**

Reorders multiple sections inside a specific task list.  
Provide an array of section UUIDs arranged in the new desired order.

---

## **Headers**

| Key | Value | Required | Description |
|------|--------|----------|-------------|
| Authorization | Token `<token>` | Yes | User authentication token |
| Content-Type | application/json | Yes | JSON request body |

---

## **Request Body**

```json
{
  "ordered_section_uuids": [
    "4937aeb4-fa3b-4ec1-9c88-cc0b9c01e85b",
    "558ba33a-6f6e-4120-bb75-119fc61013b7"
  ]
}

---
## **Successful Response**
```json
[
{
  "status": "Sections reordered successfully."
}
]
```
---
## **Response fields**

| Field  | Type   | Description                                   |
|--------|--------|-----------------------------------------------|
| status | string | Confirmation message after reordering items   |
