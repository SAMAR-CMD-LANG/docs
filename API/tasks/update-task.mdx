title: Update Task

### **PATCH /tasks/tasks/{task_uuid}**

Updates an existing task.  
This example shows moving a task to another section using `target_section_uuid`.

***

## **Headers**

| Key           | Value            | Required | Description               |
| ------------- | ---------------- | -------- | ------------------------- |
| Authorization | Token `<token>`    | Yes      | User authentication token |
| Content-Type  | application/json | Yes      | JSON request body         |

***

## **Request Body**

```json
{
  "target_section_uuid": "558ba33a-6f6e-4120-bb75-119fc61013b7"
}
```

***

## **Response Fields**

| Field        | Type        | Description                    |
| ------------ | ----------- | ------------------------------ |
| uuid         | string      | Unique identifier for the task |
| title        | string      | Title of the task              |
| description  | string      | Description text               |
| list_uuid    | string      | Parent list UUID               |
| section_uuid | string/null | Section UUID if assigned       |
| status       | string      | Task status                    |
| start_date   | string      | Task start date                |
| start_time   | string      | Task start time                |
| due_date     | string      | Due date                       |
| due_time     | string      | Due time                       |
| order        | number      | Ordering index                 |
| priority     | string/null | Priority level                 |
| is_archived  | boolean     | Whether archived               |
| reminders    | array       | List of attached reminders     |
| created_at   | string      | Creation timestamp             |
| updated_at   | string      | Update timestamp               |
