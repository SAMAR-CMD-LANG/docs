# GET `/notes/{note_uuid}/`

Retrieve a specific note using its UUID.  
Returns full metadata, content, attachments, and any AI-generated insights.

---

## üîê Required Headers

| Key | Value Format | Required | Description |
|------|-------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the request |

---

## üì• Response ‚Äî 200 OK

### Response Fields

| Field            | Type         | Description |
|------------------|--------------|-------------|
| note_uuid        | UUID         | Unique identifier for the note |
| title            | String       | Note title |
| category         | Integer      | Category ID |
| labels           | Array        | Labels tied to the note |
| is_pinned        | Boolean      | Whether note is pinned |
| is_public        | Boolean      | Whether note is public |
| content          | String       | Full content including HTML + text |
| attachments      | Array        | Note attachments (audio/image/doc) |
| ai_content_items | Array        | AI-generated insights |
| created_at       | ISO datetime | Creation timestamp |
| updated_at       | ISO datetime | Last update timestamp |
