# POST `/notes/{note_uuid}/transcribe/{attachment_id}/`

Generate a transcription from an existing **audio file** attached to a note.  
This endpoint processes the audio, extracts text, and updates the note's content with the transcription.

### Response Fields

| Field            | Type         | Description |
|------------------|--------------|-------------|
| note_uuid        | UUID         | Note identifier |
| title            | String       | Note title |
| category         | Integer      | Category ID |
| labels           | Array        | Associated labels |
| is_pinned        | Boolean      | Whether the note is pinned |
| is_public        | Boolean      | Public visibility status |
| content          | String       | Updated note content including transcription |
| attachments      | Array        | Attachment metadata |
| ai_content_items | Array        | AI insights present in the note |
| created_at       | ISO datetime | Original creation timestamp |
| updated_at       | ISO datetime | Updated timestamp after transcription |
