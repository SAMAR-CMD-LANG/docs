# POST `/notes/{note_uuid}/transcribe/{attachment_id}/`

Transcribe an existing **audio attachment** within a note.  
This endpoint processes the stored audio file, extracts speech into text, and appends the transcription directly into the note content.

### Response Fields

| Field            | Type         | Description |
|------------------|--------------|-------------|
| note_uuid        | UUID         | Note identifier |
| title            | String       | Note title |
| category         | Integer      | Category assigned to the note |
| labels           | Array        | Labels linked to the note |
| is_pinned        | Boolean      | Whether the note is pinned |
| is_public        | Boolean      | Whether the note is publicly visible |
| content          | String       | Updated HTML content including transcription |
| attachments      | Array        | List of attachments in the note |
| ai_content_items | Array        | AI insights for the note (empty if none) |
| created_at       | ISO datetime | Timestamp when note was created |
| updated_at       | ISO datetime | Timestamp when note was last updated |
