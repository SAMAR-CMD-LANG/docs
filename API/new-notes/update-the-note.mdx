# PATCH `/notes/{note_uuid}/`

Update an existing note.  
Supports updating the title, category, content, labels, and public/pinned status.

---

## üîê Required Headers

| Key | Value Format | Required | Description |
|------|-------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the user |
| Content-Type  | application/json | Yes | JSON request body |

---

## üì§ Request Body

| Field      | Type     | Required | Description |
|------------|----------|----------|-------------|
| title      | String   | No       | New title of the note |
| category   | Integer  | No       | Updated category |
| labels     | Array    | No       | Updated labels array |
| is_pinned  | Boolean  | No       | Whether the note should be pinned |
| is_public  | Boolean  | No       | Whether the note is publicly shareable |
| content    | String   | No       | Updated note content |

---

## üì• Response ‚Äî 200 OK

### Response Fields

| Field            | Type         | Description |
|------------------|--------------|-------------|
| note_uuid        | UUID         | Unique identifier |
| title            | String       | Updated note title |
| category         | Integer      | Updated category |
| labels           | Array        | Updated labels |
| is_pinned        | Boolean      | Whether the note is pinned |
| is_public        | Boolean      | Whether the note is public |
| content          | String       | Note content |
| attachments      | Array        | Attachments linked to the note |
| ai_content_items | Array        | AI-generated items |
| created_at       | ISO datetime | Creation timestamp |
| updated_at       | ISO datetime | Last updated timestamp |
