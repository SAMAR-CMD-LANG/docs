# POST `/auth/social-login/`

Authenticate a user using **Google Single Sign-On (SSO)**.  
The client must first obtain an OAuth **Access Token** and **ID Token** from Google.  
These tokens are then sent to this endpoint to verify the user and create a session in the Samantha system.

## üì§ Request Body

| Field        | Type   | Required | Description |
|--------------|--------|----------|-------------|
| provider     | String | Yes      | Must be `"google"` |
| access_token | String | Yes      | OAuth Access Token from Google |
| id_token     | String | Yes      | Google ID Token identifying the user |

## Example Request

```json
{
  "provider": "google",
  "access_token": "GOOGLE_ACCESS_TOKEN_HERE",
  "id_token": "GOOGLE_ID_TOKEN_HERE"
}
```
