---
title: "Update Label"
description: "Modify an existing note label's name or color."
---

**PATCH** `/notes/labels/{label_uuid}/`

Updates an existing label's **name** and/or **color**.  
Only authenticated users can update their own labels.

---

## üîê Required Headers

| Key           | Value Format     | Required | Description |
|--------------|-------------------|----------|-------------|
| Authorization | Token `<token>`    | Yes      | User authentication token |
| Content-Type  | application/json | Yes      | JSON request body |

---

## üìù Request Body Fields

| Field | Type   | Required | Description |
|-------|--------|----------|-------------|
| name  | string | No       | New display name for the label |
| color | string | No       | New color identifier (e.g., combo-box-green) |

---

## üìÆ Example cURL

```bash
curl -X PATCH "https://your-api.com/notes/labels/24a7aedc-975d-4f6d-8147-1cc46c6fcc93/" \
  -H "Authorization: Token <token>" \
  -H "Content-Type: application/json" \
  -d '{
        "name": "Very Important",
        "color": "combo-box-green"
      }'
```

---

# ‚úÖ Successful Response (200 OK)

```json
{
```
