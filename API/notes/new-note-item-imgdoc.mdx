---
title: "Create Note Item (Image or Document Upload)"
description: "Create a new note item inside a note by uploading an image or document. Supports optional text content."
---

**POST** `/notes/{note_uuid}/items/`

Creates a **new note item** inside an existing note using an uploaded **image** or **document** file  
(`.png`, `.jpg`, `.pdf`, `.docx`, etc.).  

The server stores the file as an **attachment** and links it to the note item.

---

## üîê Required Headers

| Key           | Value Format           | Required | Description |
|--------------|-------------------------|----------|-------------|
| Authorization | `Token <token>`        | Yes      | User authentication token |
| Content-Type  | `multipart/form-data`  | Yes      | Required when uploading files |

---

## üì§ Form-Data Fields

| Field      | Datatype           | Required | Description |
|------------|---------------------|----------|-------------|
| file       | file (image/doc)    | Yes      | The uploaded file |
| item_uuid  | string              | Yes      | UUID for the new note item |
| content    | string              | No       | Optional extra text |
| order      | number              | Yes      | Position of the item inside the note |

---

## üìù Example Form-Data (JSON representation)

```json
[
  { "key": "file", "type": "file", "value": ["Screenshot.png"] },
  { "key": "item_uuid", "type": "text", "value": "589c21db-3041-4b39-94c8-42cdb05c597b" },
  { "key": "order", "type": "text", "value": "5" },
  { "key": "content", "type": "text", "value": "" }
]

## üìÆ Example cURL

```bash
curl -X POST "https://your-api.com/notes/{note_uuid}/items/" \
  -H "Authorization: Token <token>" \
  -F "file=@Screenshot.png" \
  -F "item_uuid=589c21db-3041-4b39-94c8-42cdb05c597b" \
  -F "order=5" \
  -F "content="

```

## üì¶ Response Body Fields

| Field            | Datatype | Required | Description                              |
|------------------|----------|----------|------------------------------------------|
| item_uuid        | string   | Yes      | Unique identifier of the created note item |
| content          | string   | Yes      | Text content (may be empty)              |
| order            | number   | Yes      | Position of the item within the note     |
| attachment       | array    | Yes      | Array of attachment metadata             |
| ai_content_items | array    | Yes      | AI-generated content (if any)            |
| created_at       | string   | Yes      | Timestamp when the item was created      |
| updated_at       | string   | Yes      | Timestamp when the item was last updated |
