---
title: "Create New Note Item"
description: "Adds a new text item inside an existing note, supporting ordered content blocks and attachments."
---

**POST** `/notes/{note_uuid}/items/`

Creates a new **note item** inside an existing note.  
Each note can contain multiple items (text blocks, images, audio, etc.).  
Only the **note owner** may add items.

---

## üîê Required Headers

| Key           | Value Format      | Required | Description |
|--------------|--------------------|----------|-------------|
| Authorization | `Token <token>`   | Yes      | User authentication token |
| Content-Type  | `application/json` | Yes     | JSON request body |

---

## üìù Request Body Fields

| Field      | Datatype | Required | Description |
|-----------|----------|----------|-------------|
| item_uuid | string   | Yes      | Unique UUID you assign for the new note item |
| content   | string   | Yes      | Text/HTML content for this note item |

---

## üìå Example Request

**URL**
```
{{base_url}}notes/2fe802b9-0bc9-4f2c-99d4-d4bb9871f844/items/
```

**Request Body**
```json
{
  "item_uuid": "6e9c49af-9c09-46a0-a88c-c32084be3925",
  "content": "Additional note content"
}
```

## üìÆ Example cURL

```bash
curl -X POST "https://your-api.com/notes/2fe802b9-0bc9-4f2c-99d4-d4bb9871f844/items/" \
  -H "Authorization: Token 6a0110..." \
  -H "Content-Type: application/json" \
  -d '{
        "item_uuid": "6e9c49af-9c09-46a0-a88c-c32084be3925",
        "content": "Additional note content"
      }'
```

# üì¶ Response Body Fields

| Field            | Datatype | Required | Description |
|------------------|----------|----------|-------------|
| item_uuid        | string   | Yes      | Unique identifier of the created item |
| content          | string   | Yes      | Text content of the note item |
| order            | number   | Yes      | Position of the item inside the note |
| attachment       | array    | Yes      | List of attachments (empty if none) |
| ai_content_items | array    | Yes      | AI-generated additions linked to this item |
| created_at       | string   | Yes      | Creation timestamp |
| updated_at       | string   | Yes      | Last updated timestamp |
