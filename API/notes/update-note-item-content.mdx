---
title: "Update Note Item Content"
description: "Patch an existing note item to update its text content."
---

**PATCH** `/notes/{note_uuid}/items/{item_uuid}/`

Updates the **content** of an existing note item.  
Only the owner of the note can modify its items.  
Attachments and metadata remain unchanged.
