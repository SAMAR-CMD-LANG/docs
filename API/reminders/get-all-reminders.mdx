# GET `/tasks/tasks/{task_uuid}/reminders/`

Retrieve all reminders linked to a specific task.  
Only authenticated users can access their own reminders.

### Response Fields

| Field | Type | Description |
|--------|--------|-------------|
| uuid | UUID | Unique identifier of the reminder |
| offset_preset | String/null | Reminder preset (e.g., `"at_due_time"`) or null |
| actual_trigger_time | ISO datetime | Exact trigger time of the reminder |
| is_active | Boolean | Whether the reminder is active |
