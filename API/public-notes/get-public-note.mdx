# GET `/notes/share/{note_uuid}/`

Fetch a publicly shared note by its UUID.  
Authentication is required even for public notes.

### Response Fields

| Field | Type | Description |
|--------|--------|-------------|
| note_uuid | UUID | Unique identifier of the note |
| title | String | Title of the note |
| category | Number | Category ID associated with the note |
| labels | Array | List of labels |
| is_pinned | Boolean | Whether the note is pinned |
| is_public | Boolean | Whether the note is public |
| content | String | Main body text |
| attachments | Array | List of attached files |
| ai_content_items | Array | AI-generated content pieces |
| created_at | ISO datetime | Creation timestamp |
| updated_at | ISO datetime | Last updated timestamp |
