# POST `/notes/share/{note_uuid}/comments/`

Add a comment to a shared note.  
Only authenticated users can add comments.

---

## üîê Required Headers

| Key | Value Format | Required | Description |
|------|-------------|----------|-------------|
| Authorization | Token `<token>` | Yes | Authenticates the user |

---

## üîó Path Parameters

| Parameter | Type | Required | Description |
|-----------|--------|----------|-------------|
| note_uuid | UUID | Yes | ID of the note to add the comment to |

---

## üì§ Request Body

| Field | Type | Required | Description |
|--------|--------|----------|-------------|
| content | String | Yes | The comment text |

## Example Request Body

```json
{
  "content": "another reply to your comment"
}
```
## Example Response
```json
{
  "comment_uuid": "f1380777-69ec-4e0c-86c1-cc6c0cc655f3",
  "content": "another reply to your comment",
  "user_name": "samarjeeth RS",
  "replies": [],
  "created_at": "2025-11-19T09:07:05.623735Z",
  "updated_at": "2025-11-19T09:07:05.623764Z"
}
```
---
## Response Fields
| Field        | Type         | Description                            |
|--------------|--------------|----------------------------------------|
| comment_uuid | UUID         | Unique identifier of the comment       |
| content      | String       | Comment text                           |
| user_name    | String       | Name of the user who posted the comment |
| replies      | Array        | List of replies (empty on creation)    |
| created_at   | ISO datetime | Time the comment was created           |
| updated_at   | ISO datetime | Last updated time                      |
