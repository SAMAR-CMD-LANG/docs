Retrieve all comments for a publicly shared note.  
Authentication is required.

### Comment Object Fields

| Field | Type | Description |
|--------|--------|-------------|
| comment_uuid | UUID | Unique comment identifier |
| content | String | Comment text |
| user_name | String | Name of the user who posted the comment |
| replies | Array | Replies to this comment |
| created_at | ISO datetime | Time the comment was created |
| updated_at | ISO datetime | Last updated timestamp |
