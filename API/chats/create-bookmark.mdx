**POST** `/chats/messages/{message_uuid}/bookmark/`

This endpoint allows the authenticated user to **bookmark a specific chat message**.  
A bookmarked message can be retrieved later via the `/chats/bookmarks/` endpoint.

The message being bookmarked must:

- Belong to a session owned by the user
- Exist and be accessible to the authenticated user

No request body is required.
