**DELETE** `/fcm/register/`

This endpoint **deactivates** the user's currently registered **Firebase Cloud Messaging (FCM)** token.  
Once removed, the device will **no longer receive push notifications** until a new token is registered.

A valid authentication token is required.

This endpoint does **not** take a request body.
